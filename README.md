## Описание проекта

Проект представляет собой статическую веб-страницу, сверстанную по предоставленному макету с использованием семантической HTML-разметки. Цель работы — освоить принципы создания корректной структуры веб-страницы, работу с типографикой, валидацию кода и публикацию проекта на GitHub Pages.

## Семантические блоки страницы

1. **<header>** — шапка сайта с логотипом и навигацией
2. **<main>** — основное содержимое, разделенное на секции:
   - Секция 1: Герой / Приветствие
   - Секция 2: О нас
   - Секция 3: Услуги / Товары
   - Секция 4: Портфолио
   - Секция 5: Контакты
3. **<footer>** — подвал сайта с контактной информацией и ссылками

---

### Интерактивные состояния элементов

Добавлены CSS-состояния для интерактивных элементов интерфейса:

* `:hover` — для ссылок, кнопок и карточек товаров
* `:focus` — для ссылок, кнопок и полей ввода
* `:active` — для кнопок

Это улучшает пользовательский опыт и доступность интерфейса.

---

### Микро-анимации

* Реализована анимация плавного появления секций при загрузке страницы
* Анимации выполнены средствами CSS без использования JavaScript

## Реализация Практического занятия 4

### Выполненные требования

- Реализована адаптивная верстка с использованием `@media`-запросов
- Использована стратегия **Desktop First**
- Добавлены контрольные точки адаптивности:
  - до 1200px
  - до 992px
  - до 768px
  - до 576px
- Навигационное меню сверстано с использованием **Flexbox** и адаптируется под мобильные устройства (изменение направления элементов)
- Основные контентные блоки сверстаны с использованием **CSS Grid Layout**:
  - блок преимуществ
  - карточки товаров
  - популярные товары
- Сетки корректно перестраиваются в зависимости от ширины экрана
- Адаптированы отступы и размеры контейнеров для мобильных устройств
- Реализована адаптация форм подписки и подвала сайта
