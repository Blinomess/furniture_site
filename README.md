## Описание проекта

Проект представляет собой статическую веб-страницу, сверстанную по предоставленному макету с использованием семантической HTML-разметки. Цель работы — освоить принципы создания корректной структуры веб-страницы, работу с типографикой, валидацию кода и публикацию проекта на GitHub Pages.

## Используемые технологии

- **HTML5** — семантическая разметка
- **CSS** — базовые стили (в рамках следующего занятия)
- **Git** — контроль версий
- **GitHub Pages** — хостинг проекта

---

## Семантические блоки страницы

1. **<header>** — шапка сайта с логотипом и навигацией
2. **<main>** — основное содержимое, разделенное на секции:
   - Секция 1: Герой / Приветствие
   - Секция 2: О нас
   - Секция 3: Услуги / Товары
   - Секция 4: Портфолио
   - Секция 5: Контакты
3. **<footer>** — подвал сайта с контактной информацией и ссылками

---

### Интерактивные состояния элементов

Добавлены CSS-состояния для интерактивных элементов интерфейса:

* `:hover` — для ссылок, кнопок и карточек товаров
* `:focus` — для ссылок, кнопок и полей ввода
* `:active` — для кнопок

Это улучшает пользовательский опыт и доступность интерфейса.
